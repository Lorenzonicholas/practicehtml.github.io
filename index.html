<!DOCTYPE html>

  • "<!DOCTYPE html>" Must be the very first line in your HTML file.

  • It just informs the browser how to interpret the HTML

<html>

  • The <html> tag wraps the entire web page and it tells the browser: “Everything inside this tag is part of an HTML document.”

  • It is the root (top-level) element of any HTML page.

  • Everything else (<head>, <body>, etc.) goes inside <html>.

<body>

  • The <head> tag contains metadata — information about the web page, not shown directly on the screen.

  • <title>: Sets the browser tab title

  • <meta charset="UTF-8">: Sets character encoding (important!)

  • <meta name="viewport"...>: Mobile-friendly display
